# Awesome Agent Skills

A curated directory of useful Agent Skills, skill repositories, templates, and references.

本目录用于收集值得学习或复用的 skills 资源。收录原则是实用、可验证、结构清晰，而不是追求数量。

## Official And Reference Repositories

- [anthropics/skills](https://github.com/anthropics/skills) - Public examples, templates, and the Agent Skills specification reference.
- [google/skills](https://github.com/google/skills) - Google product and Google Cloud focused Agent Skills.
- [ComposioHQ/awesome-claude-skills](https://github.com/ComposioHQ/awesome-claude-skills) - Large curated list of Claude Skills across productivity, development, data, and automation.

## Categories

### Development And Code

- Skill authoring and validation
- Testing and debugging workflows
- Repository automation
- API and SDK reference skills

### Productivity And Knowledge Work

- Document processing
- Meeting summaries
- Research workflows
- Personal organization

### Data And Analysis

- Spreadsheet analysis
- SQL and database workflows
- Visualization and reporting
- Experiment analysis

### Automation And Integrations

- SaaS app automation
- MCP tool workflows
- Cloud platform operations
- CI and deployment workflows

## Inclusion Criteria

Resources should meet most of these criteria:

- Clear `SKILL.md` entry point or equivalent documentation.
- Clear trigger conditions for when an agent should use it.
- Public license or explicit reuse permission.
- Practical example, test scenario, or verification guidance.
- No hardcoded secrets, private endpoints, or organization-only assumptions.

## Suggest A Resource

Open an issue or pull request with:

- Resource name and URL.
- Category recommendation.
- Short description of what it helps agents do.
- Why it is useful or different from existing entries.
